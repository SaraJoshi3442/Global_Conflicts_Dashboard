# Global Conflict Intelligence Dashboard

## 📌 Overview
An interactive Excel dashboard that analyzes global conflict data, providing insights into military spending, casualties, refugee displacement, and foreign aid distribution across regions and conflicts.

## 🎯 Key Features
- KPI Cards for major conflict metrics
- Military Spending Analysis
- Casualties by Conflict
- Refugee Displacement Trends
- Foreign Aid Distribution
- Interactive Slicers and Filters

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📊 Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- Data Visualization
- Business Intelligence Reporting
